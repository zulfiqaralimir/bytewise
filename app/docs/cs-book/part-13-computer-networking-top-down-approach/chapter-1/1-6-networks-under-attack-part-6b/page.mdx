# Section 1.6 Networks Under Attack (Part 6B)

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Malware: Malicious Software</span>

In the previous part, we learned:

- The importance of network security.
- The CIA Triad.
- Types of attackers.
- Passive and Active attacks.

Now another important question arises.

> <span style={{ color: "#38bdf8", fontWeight: 700 }}>How do attackers actually compromise computers?</span>

One of the most common methods is through:

> <span style={{ color: "#fbbf24", fontWeight: 700 }}>Malware</span>

Malware is responsible for millions of cyber attacks every year.

It targets:

- Personal computers
- Smartphones
- <span style={{ color: "#f43f5e", fontWeight: 700 }}>Cloud servers</span>
- Government systems
- Banks
- Hospitals
- Industrial control systems

Understanding malware is one of the <span style={{ color: "#14b8a6", fontWeight: 700 }}>foundations of cybersecurity</span>.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>What Is Malware?</span>

The word **Malware** comes from:

```text
Malicious

+

Software
```

Definition:

> **Malware is software intentionally designed to damage, disrupt, spy on, or gain unauthorized access to computer systems.**

Unlike normal software,

malware is created with harmful intentions.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Real-Life Analogy</span>

Imagine someone gives you a gift box.

Inside the box,

instead of a gift,

there is something dangerous.

The box appears harmless,

but its contents are malicious.

Malware behaves similarly.

It often disguises itself as useful software.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Goals of Malware</span>

Different malware has different objectives.

Some malware wants to:

- Steal passwords.
- Delete files.
- Encrypt data.
- Spy on users.
- <span style={{ color: "#eab308", fontWeight: 700 }}>Display advertisements.</span>
- Take control of computers.
- Spread to other computers.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Malware Categories</span>

<span style={{ color: "#d946ef", fontWeight: 700 }}>There are many types of malware.</span>

The most important ones are:

```text
Malware

│

├── Virus

├── Worm

├── Trojan Horse

├── Spyware

├── Adware

├── Ransomware

├── Rootkit

└── Bot
```

Let's study each one.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Virus</span>

## What Is a Virus?

Definition:

> **A Virus is malware that attaches itself to another file or program and spreads when that infected file is executed.**

Notice something important.

<span style={{ color: "#0ea5e9", fontWeight: 700 }}>A virus cannot spread by itself.</span>

It requires:

- <span style={{ color: "#84cc16", fontWeight: 700 }}>A host file</span>
- <span style={{ color: "#fb7185", fontWeight: 700 }}>User action</span>

---

# Real-Life Analogy

Think of a biological virus.

A virus needs a living host.

Similarly,

a computer virus needs another program to survive.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>How Does a Virus Spread?</span>

Example:

```text
Infected File

↓

User Opens File

↓

Virus Executes

↓

Infects Other Files
```

The virus keeps attaching itself to additional files.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Characteristics of a Virus</span>

A virus usually:

- Attaches to programs.
- Requires user action.
- Replicates itself.
- Infects additional files.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Damage Caused by Viruses</span>

Viruses may:

- Delete files.
- Corrupt documents.
- Slow computers.
- Display unwanted messages.
- Disable software.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Worm</span>

Now let's study another important malware.

---

# What Is a Worm?

Definition:

> **A Worm is malware that spreads automatically across networks without requiring user interaction.**

Unlike viruses,

worms do **not** need:

- A host file.
- User execution.

<span style={{ color: "#c084fc", fontWeight: 700 }}>They spread independently.</span>

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Virus vs Worm</span>

This is one of the most common interview questions.

Virus:

```text
Needs User Action
```

Worm:

```text
Spreads Automatically
```

---

# Real-Life Analogy

Imagine one infected person.

A virus spreads only if people touch the infected person.

A worm behaves more like an airborne disease.

It spreads automatically.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Worm Propagation</span>

```text
Computer A

↓

Computer B

↓

Computer C

↓

Computer D

↓

Entire Network
```

A worm can infect thousands of computers within minutes.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Damage Caused by Worms</span>

Worms may:

- <span style={{ color: "#60a5fa", fontWeight: 700 }}>Consume bandwidth.</span>
- <span style={{ color: "#e879f9", fontWeight: 700 }}>Overload networks.</span>
- <span style={{ color: "#fde047", fontWeight: 700 }}>Slow Internet traffic.</span>
- <span style={{ color: "#93c5fd", fontWeight: 700 }}>Install additional malware.</span>

---

# Famous Example

One famous example is the:

```text
WannaCry Worm
```

It spread worldwide within hours,

infecting hospitals,

businesses,

and government organizations.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Trojan Horse</span>

Now let's study one of the <span style={{ color: "#86efac", fontWeight: 700 }}>most deceptive malware types</span>.

---

# What Is a Trojan Horse?

Definition:

> **A Trojan Horse is malware** <span style={{ color: "#fca5a5", fontWeight: 700 }}>disguised as legitimate software</span>**.**

Unlike viruses,

<span style={{ color: "#fdba74", fontWeight: 700 }}>Trojans do not replicate themselves.</span>

Instead,

<span style={{ color: "#38bdf8", fontWeight: 700 }}>they trick users into installing them.</span>

---

# Origin of the Name

The name comes from the ancient Greek story:

The **Trojan Horse**.

Greek soldiers hid inside a wooden horse.

The horse appeared harmless.

Once inside the city,

the soldiers attacked.

Computer Trojans work exactly the same way.

---

# Example

Suppose you download:

```text
Free Game.exe
```

It appears legitimate.

After installation,

it secretly:

- Steals passwords.
- Opens a backdoor.
- Installs spyware.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Characteristics of Trojans</span>

Trojans:

- <span style={{ color: "#fbbf24", fontWeight: 700 }}>Disguise themselves.</span>
- <span style={{ color: "#f43f5e", fontWeight: 700 }}>Trick users.</span>
- <span style={{ color: "#14b8a6", fontWeight: 700 }}>Do not self-replicate.</span>
- <span style={{ color: "#eab308", fontWeight: 700 }}>Often install additional malware.</span>

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Spyware</span>

Now let's study malware that <span style={{ color: "#d946ef", fontWeight: 700 }}>secretly collects information.</span>

---

# What Is Spyware?

Definition:

> **Spyware is malware that secretly monitors user activity and collects information without the user's knowledge.**

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Information Collected</span>

Spyware may collect:

- Passwords.
- <span style={{ color: "#0ea5e9", fontWeight: 700 }}>Browsing history.</span>
- Credit card numbers.
- Emails.
- Banking information.
- <span style={{ color: "#84cc16", fontWeight: 700 }}>Screenshots.</span>

---

# Real-Life Analogy

Imagine someone secretly installing cameras inside your house.

You continue your daily activities,

while someone records everything.

Spyware behaves similarly.

---

# Common Targets

Spyware often targets:

- Banking credentials.
- Social media accounts.
- Business information.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Adware</span>

Adware is another common malware type.

---

# What Is Adware?

Definition:

> **Adware is software that** <span style={{ color: "#fb7185", fontWeight: 700 }}>automatically displays unwanted advertisements.</span>

Some adware is merely annoying.

Other adware also tracks user behavior.

---

# Example

Suppose every few minutes,

your browser opens:

```text
Win a Free Phone!
```

or

```text
Congratulations!

You Are Today's Winner!
```

These advertisements may be generated by adware.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Risks</span>

Some adware:

- Slows computers.
- Tracks browsing activity.
- Installs additional malware.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Ransomware</span>

One of the most dangerous malware types today is:

> <span style={{ color: "#c084fc", fontWeight: 700 }}>Ransomware</span>

---

# What Is Ransomware?

Definition:

> **Ransomware encrypts a victim's files and demands payment to restore access.**

---

# Real-Life Analogy

Imagine someone locks your house.

Then says:

```text
Pay Me

or

You Never Enter Again.
```

Ransomware behaves the same way.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>How Ransomware Works</span>

```text
Victim Opens File

↓

Malware Executes

↓

Files Encrypted

↓

Payment Demand
```

The victim can no longer access their own files.

---

# Common Targets

Ransomware attacks:

- Hospitals.
- Banks.
- Schools.
- Businesses.
- Government agencies.

---

# Famous Example

The:

```text
WannaCry
```

attack affected more than 150 countries.

Hospitals,

factories,

and businesses were forced to stop operations.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Rootkit</span>

Now let's study one of the <span style={{ color: "#60a5fa", fontWeight: 700 }}>most difficult malware</span> types to detect.

---

# What Is a Rootkit?

Definition:

> **A Rootkit is malware designed to hide itself and other malicious software from the operating system and security software.**

<span style={{ color: "#e879f9", fontWeight: 700 }}>Its purpose is stealth.</span>

---

# Real-Life Analogy

Imagine a burglar who not only enters your house,

but also removes every trace that they were ever there.

Rootkits behave similarly.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Why Are Rootkits Dangerous?</span>

Rootkits can:

- <span style={{ color: "#fde047", fontWeight: 700 }}>Hide malware.</span>
- <span style={{ color: "#93c5fd", fontWeight: 700 }}>Hide files.</span>
- <span style={{ color: "#86efac", fontWeight: 700 }}>Hide processes.</span>
- <span style={{ color: "#fca5a5", fontWeight: 700 }}>Hide network activity.</span>

<span style={{ color: "#fdba74", fontWeight: 700 }}>They often allow attackers to remain inside systems for long periods.</span>

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Bot</span>

Now let's study another important malware component.

---

# What Is a Bot?

Definition:

> **A Bot is malware that allows an attacker to remotely control an infected computer.**

The infected computer becomes a:

```text
Zombie Computer
```

<span style={{ color: "#38bdf8", fontWeight: 700 }}>because it follows commands from the attacker.</span>

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>How Bots Work</span>

```text
Attacker

↓

Command Server

↓

Bot

↓

Victim Computer
```

The attacker controls the infected computer remotely.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Botnet</span>

A collection of bots is called a:

> <span style={{ color: "#fbbf24", fontWeight: 700 }}>Botnet</span>

Definition:

> **A Botnet is a network of compromised computers controlled by a single attacker.**

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Botnet Diagram</span>

```text
             Attacker

                 │

        Command Server

      ┌──────┼──────┐

      ▼      ▼      ▼

 Computer Computer Computer

      ▼      ▼      ▼

     Bot     Bot     Bot
```

Thousands or even millions of infected computers can participate.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>What Are Botnets Used For?</span>

Botnets are commonly used for:

- <span style={{ color: "#f43f5e", fontWeight: 700 }}>DDoS attacks.</span>
- <span style={{ color: "#14b8a6", fontWeight: 700 }}>Spam emails.</span>
- <span style={{ color: "#eab308", fontWeight: 700 }}>Cryptocurrency mining.</span>
- Password attacks.
- Malware distribution.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>How Malware Spreads</span>

Malware reaches victims in many ways.

Common methods include:

- Email attachments.
- Fake software downloads.
- Malicious websites.
- USB drives.
- Software vulnerabilities.
- Fake mobile apps.
- Social engineering.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Social Engineering</span>

Sometimes,

attackers do not attack computers first.

They attack:

> <span style={{ color: "#d946ef", fontWeight: 700 }}>People</span>

Example:

An email claims:

```text
Urgent!

Reset Your Password Immediately!
```

The victim clicks the link.

Malware installs automatically.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Malware Prevention</span>

Several techniques reduce malware risk.

Examples include:

- Antivirus software.
- Regular software updates.
- Firewalls.
- Strong passwords.
- Multi-Factor Authentication (MFA).
- Downloading software only from trusted sources.
- Avoiding suspicious email attachments.
- <span style={{ color: "#0ea5e9", fontWeight: 700 }}>Regular data backups.</span>

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Comparison of Malware Types</span>

| Malware | Self-Replicates | Requires User Action | Main Purpose |
|----------|----------------|----------------------|--------------|
| Virus | Yes | Yes | Infect files |
| Worm | Yes | No | Spread across networks |
| Trojan Horse | No | Yes | Disguise itself as legitimate software |
| Spyware | No | Usually Yes | Monitor user activity |
| Adware | No | Usually Yes | Display unwanted advertisements |
| Ransomware | No | Usually Yes | Encrypt files and demand payment |
| Rootkit | No | Often Yes | Hide malware from detection |
| Bot | No | Usually Yes | Allow remote control of infected systems |

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Common Misconceptions</span>

## Misconception 1

> Every malware is a virus.

Incorrect.

A virus is only one type of malware.

---

## Misconception 2

> Worms require users to open infected files.

Incorrect.

Worms spread automatically across networks.

---

## Misconception 3

> Trojans replicate like viruses.

Incorrect.

Trojans do not self-replicate.

They rely on deception.

---

## Misconception 4

> Antivirus software catches every malware.

Incorrect.

New or sophisticated malware may evade detection.

<span style={{ color: "#84cc16", fontWeight: 700 }}>Good security requires multiple layers of protection.</span>

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Key Takeaways</span>

- Malware is software designed to harm, exploit, or gain unauthorized access to systems.
- A Virus attaches to another program and requires user action to spread.
- A Worm spreads automatically without user interaction.
- A Trojan Horse disguises itself as legitimate software.
- Spyware secretly collects user information.
- Adware displays unwanted advertisements and may track user behavior.
- Ransomware encrypts files and demands payment.
- Rootkits hide malware from the operating system and security software.
- Bots allow attackers to remotely control infected computers.
- A Botnet is a collection of compromised computers controlled by a single attacker.

---

# <span style={{ color: "#a5b4fc", fontWeight: 700 }}>Technical Vocabulary</span>

| Term | Definition |
|------|------------|
| Malware | Malicious software designed to damage, disrupt, or gain unauthorized access to computer systems. |
| Virus | Malware that attaches itself to another file or program and spreads when the host is executed. |
| Worm | Malware that spreads automatically across networks without user interaction. |
| Trojan Horse | Malware disguised as legitimate software to deceive users. |
| Spyware | Malware that secretly monitors user activity and collects sensitive information. |
| Adware | Software that displays unwanted advertisements, sometimes while collecting user data. |
| Ransomware | Malware that encrypts a victim's files and demands payment for decryption. |
| Rootkit | Malware designed to hide itself and other malicious software from detection. |
| Bot | Malware that allows remote control of an infected computer. |
| Botnet | Network of compromised computers controlled by a single attacker. |
| Social Engineering | Psychological manipulation used to trick people into revealing information or performing unsafe actions. |
