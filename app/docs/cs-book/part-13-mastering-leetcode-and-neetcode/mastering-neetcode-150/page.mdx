# How We'll Work Together

For **each problem**, the learning process will follow a structured approach:

## 1. Problem Statement
- A clear description of the problem.

## 2. Constraints
- Important limits and conditions to consider.

## 3. Hint (Optional)
- A gentle hint to guide your thinking before viewing the solution.

## 4. Your Attempt
- Space for you to solve the problem on your own.

## 5. Complete Walkthrough

The complete walkthrough will include:

- **Intuition:** Understand the core idea behind the solution.
- **Approach:** Learn the step-by-step algorithm and reasoning.
- **Code (Python):** Write a clean, interview-quality Python solution.
- **Complexity Analysis:** Analyze the **Time Complexity** and **Space Complexity** of the solution.
- **Edge Cases:** Identify and handle inputs that break naive solutions.

In short, every problem will be explained as:

> **Intuition → Approach → Code (Python) → Complexity Analysis → Edge Cases**
