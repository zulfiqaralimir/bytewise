# Mastering Claude Code

Claude Code is two things at once:

- **A CLI tool** — an AI pair programmer that lives in your terminal and can read, write, run, and reason about your entire codebase
- **An API platform** — a set of endpoints and SDKs that let you build your own AI-powered tools, agents, and products

This part covers both, in sequence. You will leave knowing how to *use* Claude Code like a power user, and how to *build with* the Claude API like a developer.
